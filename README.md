# RoundLock (CEP Extension)

RoundLock is a CEP extension for Adobe After Effects that gives you full control over Rectangle Paths, anchoring, and Layer organization. It solves a common issue in AE: keeping the **Roundness** intact while freely editing the **Size** of a shape.

The extension offers a visual 3√ó3 anchor selector, automatic expression setup, and a complete Lottie optimization toolkit to clean up shape layers in seconds.

### üîµ Anchor Point Lock System

  - Auto-compensate the **Position** so the shape stays perfectly in place
- Freely edit **Size** without losing **Roundness**
- Works with Rectangle Path inside shape layers

### Anchor Lock

2. Open **RoundLock**.
3. Click any of the 9 anchor point icons.
4. That‚Äôs it ‚Äî you can now resize the Rectangle Path without breaking its Roundness.

### Lottie Optimizer
1. Select your Shape Layer(s).
2. Open the **Lottie Optimizer** section.
3. Scan your layer to detect:

   - Fills / Strokes / Gradients
